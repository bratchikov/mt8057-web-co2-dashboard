# CO2 Dashboard Tray Application

Приложение для мониторинга уровня CO2 и температуры в системном трее Windows.

## Обзор

CO2 Dashboard Tray - это легкое приложение для Windows, которое отображает текущий уровень CO2 и температуру в системном трее. Приложение получает данные от Go backend сервера и визуально отображает состояние с помощью цветных иконок со стрелками направления.

### Особенности

- Отображение уровня CO2 в трее Windows
- Цветовая индикация состояния (зеленый, оранжевый, красный)
- Стрелка направления изменения (рост, стабильно, падение)
- Настройка порогов и интервала опроса
- Логирование ошибок подключения

## Требования

- Windows 10/11
- .NET 10 Desktop Runtime
- Go backend сервер на `localhost:8072`

## Сборка

```bash
cd CO2DashboardTray
dotnet publish -c Release -o publish
```

## Запуск

```bash
cd publish
dotnet CO2DashboardTray.dll
```

Или просто запустите `CO2DashboardTray.exe` из папки `publish`.

## Настройка

Приложение использует файл `appsettings.json` для хранения настроек:

```json
{
  "ServerUrl": "http://localhost:8072",
  "PollingIntervalSeconds": 60,
  "NormalThreshold": 1000,
  "WarningThreshold": 1500
}
```

### Параметры настроек

| Параметр | Описание | Значение по умолчанию |
|----------|----------|----------------------|
| `ServerUrl` | URL сервера Go backend | `http://localhost:8072` |
| `PollingIntervalSeconds` | Интервал опроса сервера (сек) | `60` |
| `NormalThreshold` | Порог нормального CO2 (ppm) | `1000` |
| `WarningThreshold` | Порог предупреждения CO2 (ppm) | `1500` |

## Архитектура

### Компоненты приложения

| Компонент | Описание |
|-----------|----------|
| `Program.cs` | Точка входа в приложение |
| `TrayApp.cs` | Основной класс управления tray-приложением |
| `Settings.cs` | Модель настроек |
| `SettingsManager.cs` | Менеджер загрузки/сохранения настроек |
| `DataClient.cs` | HTTP клиент для API сервера |
| `IconManager.cs` | Генерация иконок в зависимости от состояния CO2 |

### Цветовая индикация

- **Зеленый** (≤ NormalThreshold): Нормальный уровень CO2
- **Оранжевый** (NormalThreshold < CO2 ≤ WarningThreshold): Предупреждение
- **Красный** (> WarningThreshold): Высокий уровень CO2
- **Серый**: Сервер недоступен

### Стрелки направления

- **↗** (U+2197): Рост уровня CO2
- **→** (U+2192): Стабильный уровень
- **↘** (U+2198): Падение уровня CO2

## Логирование

Ошибки подключения и другие события логируются в консоль. При запуске через `dotnet` сообщения будут отображаться в терминале.

## Сборка без SDK

Для распространения приложения можно использовать self-contained сборку:

```bash
dotnet publish -c Release -o publish --self-contained true -r win-x64
```
