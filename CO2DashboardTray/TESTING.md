# Инструкция по тестированию CO2 Dashboard Tray

## Тест подключения к серверу

### Шаги

1. Убедитесь, что Go backend сервер запущен на `localhost:8072`
2. Запустите приложение:
   ```bash
   cd CO2DashboardTray/publish
   dotnet CO2DashboardTray.dll
   ```
3. Проверьте, что иконка отображается в системном трее
4. Наведите курсор на иконку, чтобы увидеть tooltip

### Ожидаемый результат

- Иконка отображается в трее
- Tooltip показывает формат: `CO2: XXX ppm (стрелка) | Температура: XX.X°C`
- Цвет иконки соответствует уровню CO2

## Тест ошибок подключения

### Шаги

1. Остановите Go backend сервер
2. Подождите интервал опроса (по умолчанию 60 секунд)
3. Проверьте иконку в трее

### Ожидаемый результат

- Иконка становится серой (кружок без стрелки)
- Tooltip показывает "Сервер недоступен"
- При возобновлении соединения иконка обновляется корректно

## Тест смены значений CO2

### Шаги

1. Запустите приложение
2. Проверьте направление стрелки при изменении CO2:
   - Рост CO2 → стрелка ↗
   - Стабильный CO2 → стрелка →
   - Падение CO2 → стрелка ↘
3. Проверьте цвет иконки при переходе через пороги:
   - ≤1000 ppm → зеленый
   - 1001-1500 ppm → оранжевый
   - >1500 ppm → красный

### Ожидаемый результат

- Стрелка меняет направление при изменении CO2
- Цвет иконки меняется при переходе через пороги

## Тест настроек

### Шаги

1. Откройте контекстное меню иконки (правый клик)
2. Выберите "Настройки"
3. Измените параметры в `appsettings.json`
4. Перезапустите приложение
5. Проверьте, что изменения применены

### Ожидаемый результат

- Файл `appsettings.json` открывается в редакторе по умолчанию
- Изменения применяются после перезапуска

## Тест DPI масштабирования

### Шаги

1. Измените масштаб экрана в Windows (Параметры > Система > Экран > Масштабирование)
2. Проверьте иконки при разных значениях (100%, 125%, 150%, 200%, 250%, 300%)
3. Убедитесь, что иконки не размываются

### Ожидаемый результат

- Иконки корректно отображаются при разных значениях DPI
- Текст и стрелки четкие, без размытия

## Тест контекстного меню

### Шаги

1. Правый клик по иконке в трее
2. Проверьте пункты меню:
   - "Настройки" - открывает файл appsettings.json
   - "Выход" - завершает приложение

### Ожидаемый результат

- Оба пункта меню работают корректно
- Приложение корректно завершается при выборе "Выход"

## Тест обработки ошибок

### Шаги

1. Укажите неверный URL сервера в `appsettings.json`
2. Запустите приложение
3. Проверьте, что приложение не падает

### Ожидаемый результат

- Приложение продолжает работать
- Отображается серая иконка (кружок без стрелки)
- Ошибки логируются в консоль

## Тест повторного запроса при ошибке

### Шаги

1. Отключите сервер
2. Подождите несколько интервалов опроса
3. Включите сервер обратно
4. Проверьте иконку

### Ожидаемый результат

- При отключении сервера отображается серая иконка
- При восстановлении соединения иконка обновляется корректно
